# Todo app
<p>Todo is a web application for creating task lists in the browser. Users can add, delete, mark tasks as completed and filter tasks.</p>

# Technologies used
<ul>

  <li>Sass(SCSS)</li>
  <li>Vue.js</li>
</ul>


# Launch Instructions
```
git clone https://github.com/Horizon-git/vue-todo-app.git
cd project_folder
npm install
npm start
```
